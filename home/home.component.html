Campfire is built, for you

When we built Campfire, we built it around the sole idea of simplicity and getting things done.

Today, this ideaology is still heavily present throughout all of our products, as we continue our mission to build the greatest Discord bot, for you.

We not only built Campfire for ease, but we built it with you (the user) in mind. That's why we pioneer to make everything controllable via our web dashboard.

Always online

Hosted on Amazon Servers

Campfire is hosted on fast and premium cloud computing services, powered by AWS. This means that we have almost 100% uptime. We make sure the bot stays online, so you can have peace of mind knowing that it always will be!

What's more? We have backup servers from Digital Ocean and Google Cloud in case anything goes wrong.

Built with Node

Campfire is built with Discord.js for the best user experience and is powered by Chromium's V8 engine via Node.js.
